# python-fullstack-task1-maincrafts
PYTHON FULL STACK WEB DEVELOPMENT – TASK 1 

Technologies Used:
Layer Technology
Frontend HTML, CSS 
Backend Python, Flask
Database SQLite 
Tools VS Code, Browser

Application Features:
Home page with user input form

Form submission handled by Flask backend

User data stored in SQLite database

Fetch and display all users in table format

Clean and simple user interface

Project Structure: 
PYTHON-FULL-STACK-TASK-1/ │ ├── app.py # Flask backend logic ├── templates/ │ └── index.html # Frontend UI ├── static/ │ └── style.css # Styling ├── database.db # SQLite database └── README.md # Project documentation

Project Flow:
User enters details in the HTML form

Form data is sent to the Flask server

Flask processes the request

Data is stored in SQLite database

Stored user details are retrieved

All users are displayed on the web page
